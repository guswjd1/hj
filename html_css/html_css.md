## HTML Basic

- 기본구조

※ ` : backick

```
<!DOCTYPE html>
<html>
  <head>
  웹 문서의 부가정보
  </head>
  <body>
  웹 문서의 내용
  </body>
</html>
```

## HTML Element

- Tag와 contents로 구성
- Tag는 시작태그와 종료태그로 구성

  - contents와 종료태그 없이 시작태그만 있는 요소 : 빈요소(Empty Element)

  ```
  <h1>제목</h1>

  <br> → 빈요소 (Empty Element)
  ```
